🎓 Student Placement Tracking System

A web-based application developed to manage and track student placement activities in an efficient and structured manner.
This system helps colleges maintain student records, company details, and placement status in a centralized platform.

🚀 Features <br>
👨‍🎓 Student Registration & Management (CRUD operations)<br>
🏢 Company Recruitment Management<br>
📊 Placement Status Tracking (Placed / Not Placed / In Progress)<br>
📅 Interview Scheduling & Updates<br>
🔍 Search & Filter Student/Company Records<br>
🗂️ Centralized Database Management<br>
🧭 MVC Architecture-based design<br>
🔐 Secure database connectivity using JDBC<br>
🛠️ Tech Stack<br>
💻 Core Java<br>
🌐 Servlets<br>
🔗 JDBC<br>
🗄️ MySQL<br>
🎨 HTML, CSS, Bootstrap<br>
⚙️ JavaScript<br>
🧱 MVC Architecture<br>
🚀 Apache Tomcat Server<br>
