🎓 Student Placement Tracking System

A web-based application developed to manage and track student placement activities in an efficient and structured manner.
This system helps colleges maintain student records, company details, and placement status in a centralized platform.

🚀 Features
👨‍🎓 Student Registration & Management (CRUD operations)
🏢 Company Recruitment Management
📊 Placement Status Tracking (Placed / Not Placed / In Progress)
📅 Interview Scheduling & Updates
🔍 Search & Filter Student/Company Records
🗂️ Centralized Database Management
🧭 MVC Architecture-based design
🔐 Secure database connectivity using JDBC
🛠️ Tech Stack
💻 Core Java
🌐 Servlets
🔗 JDBC
🗄️ MySQL
🎨 HTML, CSS, Bootstrap
⚙️ JavaScript
🧱 MVC Architecture
🚀 Apache Tomcat Server
